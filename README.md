# Codigo Penal

This repository contains the Codigo Penal del Peru site and the helper scripts used
to extract and organize the source content.

Production deploy: https://codigopenalpe.netlify.app/

## Structure

- `codigo-penal-ui/` Astro site for the public web UI.
- `scripts/` Data extraction and content preparation scripts and artifacts.
- `notes/` Project notes.
